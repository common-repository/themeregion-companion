== Description ==

This plugin comes with all the core libraries of themeregion themes. Like metabox framework, theme framework & other features like widgets etc are included within this plugin. Its not for regular usage as it's meant to be used by only themeregion's themes.

== Installation ==

1. Upload `themeregion-companion.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
